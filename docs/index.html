Thermal-Quantum Numerov Solver#

A production-grade Python library for solving the 1D Time-Independent Schr√∂dinger Equation and simulating identical-particle thermalization in the Canonical Ensemble ‚Äî refactored from a legacy research notebook into a SOLID-compliant, fully documented scientific library.

Key Features#

  • Matrix Numerov solver ‚Äî \(O(dx^4)\) global accuracy via a generalised eigenvalue problem \(A\boldsymbol{\psi}=\varepsilon B\boldsymbol{\psi}\).

  • Four potential geometries ‚Äî Infinite/Finite Square Wells, Truncated Harmonic, and V-Shaped (linear) potentials.

  • Canonical Ensemble thermodynamics ‚Äî Boltzmann-weighted single-particle and two-particle densities with configurable temperature.

  • All five particle statistics ‚Äî Boltzmann, Boson, Fermion, Spin-1/2, and Spin-1 two-particle symmetry rules; Pauli exclusion enforced exactly.

  • Publication-quality plots ‚Äî energy-level diagrams, thermal density curves, 2-D heatmaps, and 3-D surface renderings via QuantumPlotter.

  • Immutable configuration ‚Äî frozen dataclasses guarantee reproducibility across every solver and plotter call.

Quick Start#

from src.models import FiniteSquareWell
from src.core  import NumerovSolver, ThermalEngine
from src.models.statistics import ParticleType
from src.visualization import QuantumPlotter

system  = NumerovSolver().solve(FiniteSquareWell(v0=50.0), n_states=20)
rho     = ThermalEngine().calculate_thermal_density(system, temperature=5.0)
rho2p   = ThermalEngine().calculate_pair_density(system, 5.0, ParticleType.FERMION)

plotter = QuantumPlotter(figsize=(9, 5), dpi=150)
plotter.plot_wavefunctions(system, n_states=5)
plotter.plot_pair_density_heatmap(system, rho2p, title="Fermion pair density")
